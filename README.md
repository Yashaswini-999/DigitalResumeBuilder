Digital Resume Builder

Description

A Java-based resume builder application that allows users to create professional and informal resumes dynamically in HTML format. Users can enter their personal, educational, and professional details through a simple console interface, and the system generates resumes using multiple templates.

This project demonstrates Java programming, file handling, and integration with HTML and CSS.

Features

- Input personal details, education, skills, experience, projects, and hobbies  
- Generate resumes dynamically in clean and formatted HTML files  
- Choose between multiple resume templates:  
  - Template 1: Informal, colorful style  
  - Template 2: Professional, clean style  
- All generated resumes are saved automatically in the `output/` folder  
- Simple console-based interaction designed for beginners  
- Optional CSS styling included to enhance the layout and appearance  

Tech Stack

- Frontend: HTML, CSS  
- Backend: Java (Core Java)  
- Database: Not required (HTML files generated and stored locally)
